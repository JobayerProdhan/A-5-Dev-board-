Question-1: What are the different way to select an element in the DOM?

Different ways to select an Element in the DOM

  1. getElementById(id)
  2. getElementsByClassName(className)
  3. getElementsByTagName(tagName)
  4. querySelector(selector)
  5. querySelectorAll(selector)
  6. closest(selector)

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Difference between innerHTML, innerText, and textContent

Property Description Use Case
innerHTML Access inner HTML of an element Useful when we want to manipulate inner HTML of HTML element
innerText Access inner Text of an element Useful when you need the text content of an element (for example: to read user-visible content).
textContent Access text content but not any HTML tags. Useful for getting text content without worrying about visible or hidden elements.

Question-4: What is event bubbling in the DOM?

In a word event bubbling is a concept of JS. It is a event that occurs on a child element 'bubbles up to it's parent element.

Question-5: How do you create, add, and remove elements using JavaScript?

  • Create an Element: Use JavaScript to create an HTML element (like a div, paragraph, etc.).
  • Add an Element: Attach the created element to the existing document or any specific container in the DOM.
  • Remove an Element: Remove an element from the DOM by accessing its parent and calling the remove function.
